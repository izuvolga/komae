まだプレビューエリアのマウスホイールによるスクロール機能が十分ではないように見える。
実際に使ってみると、白いキャンパスが全部表示されている状態でマウスホイールを回すと、スクロールができないのに違和感がある。

以下の条件を満たす場合は、スクロールできるようにしたい。

・白いキャンパスの上の辺が、チェッカーボードエリアの下の辺と重なるまで
・白いキャンパスの下の辺が、チェッカーボードエリアの上の辺と重なるまで

現在: この状態だとスクロールできない
理想: この状態だとスクロールできる
```
+--------------------+
|                    |
|   +---------+      |
|   |    A    |      |
|   |    B    |      |
|   +---------+      |
|                    |
+--------------------+
```

下にスクロールすると、白いキャンパスの上の辺がチェッカーボードエリアの下の辺と重なるギリギリまで
```
+--------------------+
|                    |
|                    |
|                    |
|   +---------+      |
|   |    A    |      |
+--------------------+
```

上にスクロールすると、白いキャンパスの下の辺がチェッカーボードエリアの下の辺と重なるギリギリまで
```
+--------------------+
|   |    B    |      |
|   +---------+      |
|                    |
|                    |
|                    |
+--------------------+
```
